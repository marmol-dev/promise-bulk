Class PromiseBulk<Input, Output>

This is the main promise-bulk feature. This class allows to instantiate bulk processes.

A bulk function is the only required constructor parameter.

Type parameters

  • Input = unknown

    The type of the input items to be processed by the bulk function.

  • Output = unknown

    The type of the response items of the bulk function.
